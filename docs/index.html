SKILLS

HTML5/CSS3

独学で学び始め、コーディングの楽しさを感じた最初の入り口でした。Webページ上で自分の書いた、CSSのアニメーションやレスポンシブの対応が実現した時は感動を覚えました。プログラミングスクール入学の事前課題として一般的なフォームの写経を行いました。課題ではコーポレートサイトの写経を行いました。

PHP

プログラムを本格的に組んだ最初の言語でした。よくわかるPHPの教科書の教科書を3周読んだのち、条件分岐や配列操作のプログラムを作成しました。簡易版Twitterのような掲示板を作成しました。ログイン機能、ログアウト機能、投稿機能、返信機能、投稿詳細表示機能、いいね機能、RT機能を実装しました。

CakePHP

オークションサイトを作成しました。ログイン機能、ユーザー管理機能、入札機能、落札機能、出品機能、メッセージ取引機能を実装しました。MVCフレームワークのCakePHPの規則に則りWebアプリケーションの開発が可能です。規則が厳しいという理由もあり最初の学習するフレームワークとしてCakePHPを選びました。

JavaScript

jQueryなどのライブラリも学びました。CakePHPで作成したオークションサイトに残り時間のタイマー表示機能や、Ajaxを用いて天気予報APIから、プルダウンで選択した地域の天気を非同期通信で取得し画面に表示させる実装をしました。

MySQL

データベースの基礎から学び、MySQLのデータベースを使用して、DDL•DML•DCLによる基本操作です。小規模なアプリケーションの概念設計•論理設計•物理設計が可能です。

AWS
